# ETCDmxTool
## Linux

### Hardware support
| Hardware      | Linux Support |
|---------------|---------------|
| USB Whip      | Yes           |
| Gadget I      | Not yet       |
| Gadget II     | Not yet       |

### Permissions
Read/write access is required to the relevant USB bus device.
For systems that support udev, this can be applied by copying "99-etcusb.rules" to /etc/udev
